Tetris Game

How to Play

Control the falling blocks to create complete horizontal lines. When a line is complete, it will disappear and you'll earn points!

Mobile Support: On-screen controls are available for touch devices. The game also adapts to different screen sizes.

Controls:

Key/Button Action
← / Left Button Move left
→ / Right Button Move right
↓ / Down Button Move down (soft drop)
↑ / Up Button Rotate piece
Space / ▼ Button Hard drop
P / ⏸ Button Pause/Resume game
R Restart game (when game over)

Scoring:

  • 1 line: 100 × level points
  • 2 lines: 300 × level points
  • 3 lines: 500 × level points
  • 4 lines: 800 × level points

The game speeds up as you level up. You gain a level for every 10 lines cleared.
