Tetris Game

How to Play

Control the falling blocks to create complete horizontal lines. When a line is complete, it will disappear and you'll earn points!

Mobile Support: Large, colorful on-screen buttons are available for touch devices. Each button has a clear icon showing its function. The game also adapts to different screen sizes.

Controls:

Button Action
← (Left arrow) Move left
→ (Right arrow) Move right
↓ (Down arrow) Move down (soft drop)
↑ (Up arrow) Move up (not used in gameplay)
↻ (Rotate button) Rotate piece
▼ (Drop button) Hard drop
⏸ (Pause button) Pause/Resume game
↺ (Restart button) Restart game (when game over)

Scoring:

  • 1 line: 100 × level points
  • 2 lines: 300 × level points
  • 3 lines: 500 × level points
  • 4 lines: 800 × level points

The game speeds up as you level up. You gain a level for every 10 lines cleared.
